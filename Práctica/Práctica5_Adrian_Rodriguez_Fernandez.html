¿Qué es Break The Chains?

Vamos implementar un sistema de block chain para crear una base de datos global en el que se recojan, datos de empresas que se quieran asociar con nosotros. De tal manera que estos datos, sobre cifras de contratación, porcentajes de cargos directivos puedan ser visibles para cualquier entidad, y cualquier usuario. La tecnología de block chain implica un bajo coste, transparencia, seguimiento de los datos y una gran seguridad en estos.
